## Overview
A light, editorial-minimal system built on an exposed drafting grid: hairline rules run the full viewport width and height, turning every section into a visible cell. Off-white warm-gray canvas, near-black ink, and elegant serif display type ("Pricing", "Blog", "Enter a new age of knowledge work") contrast with grotesque UI text and realistic spreadsheet screenshots. Color is rationed: the interface stays monochrome while the product UI inside screenshots carries the indigo brand blue. Dark near-black blocks (home hero, security section, footer) invert the scheme without changing the structure.

## Layout
Sticky top nav on the canvas color: logo left, centered link row (Solutions/Product with chevrons, Templates, Careers, Blog, Pricing), Log in + boxed Sign up right. Content sits in a center column whose left/right boundary rules extend to the page edges; horizontal hairlines separate every band, including deliberate empty spacer cells between sections. Heroes are split: serif headline + short grotesque subcopy + paired CTAs left, product screenshot or steel-blue gradient panel right (home inverts to a dark hero). Interior pages open with a huge lone serif title in its own bordered band. Sections alternate 2- and 3-column bordered cells; templates page is a strict 3-up card grid. Every page closes with the same dark footer: big white serif tagline, white "Request a Demo →" button, 5-column link columns divided by hairlines.

## Components
Buttons: primary is a near-black (#101319) rectangle with white grotesque label and 2-4px radius ("Try it now", "Sign up"); secondary is a 1px black-outlined transparent rectangle ("Request a Demo →"); on dark footer the fill flips to white with black text. Cards: flat, square-cornered, defined only by 1px hairline borders on the shared grid — no shadows, no fills beyond canvas/white. Job/tag chips ("San Francisco", "In Person", "Full Time") are 1px-outlined small-radius rectangles. Template and feature cards embed high-fidelity spreadsheet mockups with the indigo logo mark, blue links, and colored data pills. Blog cards pair engraved moiré line-art thumbnails with a boxed ↗ arrow icon. Pricing is two bordered columns with icon-tile feature rows, no highlighted plan.

## Signature
- Exposed full-bleed hairline grid: section borders extend past the content column to the viewport edge, like an architectural blueprint.
- Serif display / grotesque body split — oversized lone serif page titles and the recurring serif footer tagline "Enter a new age of knowledge work".
- Near-total monochrome chrome; the only saturated color is the indigo #4740DB living inside product screenshots and the logo.
- Black-and-white engraved/moiré line-art imagery (blog covers, testimonial panels) instead of photography or 3D renders.

## Do / Don't
Do:
- Draw 1px hairline rules that run edge-to-edge and let empty bordered spacer bands breathe between sections.
- Reserve the serif exclusively for large display moments (page titles, footer tagline); set everything functional in the grotesque.
- Keep chrome monochrome and let color appear only inside embedded product-UI screenshots (indigo accents, data pills).
- Use square or barely-rounded rectangles for buttons and chips, filled black or 1px-outlined.
- End every page with the dark footer block: serif tagline left, description + white demo CTA right, hairline-divided link columns.

Don't:
- Add drop shadows, gradients on cards, or rounded 12px+ corners — surfaces are flat and sharp.
- Use the indigo blue for page CTAs; the main buttons are black/white, blue belongs to the product UI.
- Introduce photography or illustration styles beyond the engraved black-and-white line art and steel-blue gradient panels.
- Bold the serif headlines or set them small — they work at large sizes in regular weight.
- Fill the layout wall-to-wall; the design depends on generous empty grid cells and wide side margins.
