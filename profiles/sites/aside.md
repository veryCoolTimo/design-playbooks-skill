## Overview
Airy minimal-light landing page built almost entirely from white space, black type, and one photographic sky-blue hero. The visual language mimics macOS: soft app-window mockups with traffic-light dots, subtle borders, and system-style UI chrome serve as the primary imagery. Color arrives only through product screenshots and pastel gradient card backgrounds; the site chrome itself is strictly black-on-white.

## Layout
Sticky top nav: logo left, four center text links, single black pill "Download" button right, on a white bar with a hairline divider once scrolled. Hero is a full-bleed sky/clouds photo with a small pill badge ("Backed by Y Combinator"), a two-line oversized headline in black, one black pill CTA, and a large floating browser mockup anchored to the bottom edge. Body sections sit on white inside a subtly inset content column with hairline side rules; rhythm alternates centered mega-headlines ("Anything you do in a browser, Aside can do for you.") with left-aligned two-column blocks (small blue eyebrow link left, paragraph right). Very generous vertical whitespace (200px+) between sections; dotted hairline separators between some blocks. Feature evidence appears as a 3-column card grid of pastel-gradient screenshot cards with caption text below each card.

## Components
- Buttons: solid black pill, white text, medium size; hero CTA carries a small download icon. No visible outline/ghost variants.
- Badges/eyebrows: light pill chip with thin border and chevron (hero badge); section eyebrows are small blue (#0A93E8) links with chevron arrow.
- Cards: rounded ~12-16px, soft pastel gradient backgrounds (pink, aqua, lime, mint), containing floating mini-UI screenshots with soft drop shadows; caption below is bold lead-in phrase then gray sentence run-on in the same line.
- Data viz: horizontal benchmark bar chart — one saturated blue (#1E6EF5) bar for Aside, flat light-gray bars for competitors, hairline row dividers; segmented pill tab control below.
- Browser mockups: white/near-white windows, macOS traffic lights, sidebar with favicon list, chat bubbles in light gray rounded rectangles.

## Signature
- Photographic blue-sky hero with clouds as the only large color moment on an otherwise monochrome page.
- Product-as-imagery: real macOS-style browser windows and mini-UI fragments instead of illustrations or stock art.
- Bold lead-in + gray continuation copy pattern ("Signing in Aside signs in and works across...") used for every caption and paragraph.
- Single-blue accent discipline: blue appears only in eyebrow links, text links, and the hero chart bar.

## Do / Don't
Do:
- Keep the page monochrome black-on-white and let screenshots and one hero image carry all color.
- Use pill-shaped solid black buttons for every CTA, including the nav.
- Set headlines very large in a plain grotesque at regular/medium weight, sentence case with terminal periods.
- Introduce sections with a small blue chevron-link eyebrow above or beside the content.
- Wrap feature proof in pastel-gradient rounded cards showing floating UI fragments with soft shadows.

Don't:
- Don't add saturated brand color to backgrounds, nav, or section fills — blue is reserved for links and one chart bar.
- Don't use heavy borders or hard shadows; keep hairline dividers and diffuse elevation only.
- Don't use icon-heavy feature grids or illustration; imagery is always real product UI or app favicons.
- Don't uppercase or letterspace headings; the type system relies on size contrast, not weight or caps.
- Don't crowd sections — vertical whitespace between blocks is a core part of the rhythm.
