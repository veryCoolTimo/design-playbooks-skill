## Overview
A big-type, poster-style landing page built from full-bleed flat color bands: acid yellow, pale lavender, sky blue, violet, and pink stacked top to bottom. Nearly everything is near-black type at extreme sizes — either ultra-bold condensed uppercase manifestos or enormous plain lowercase grotesque section words. The pixelated/glitched "CONTROL" logotype and a single electric-green CTA are the only ornaments; no gradients, no photography outside product screenshots.

## Layout
Sparse top bar (tiny "Publishing" left, "Tutorials / Login" right) over a centered hero: glitch-pixel logotype, small condensed uppercase tagline, one black rectangular CTA, then a browser-frame product screenshot bleeding off the fold. Sections are horizontal color bands, several separated by thin black hairline rules. "Design" / "Animate" bands pair a giant left-aligned lowercase word with a right-column screenshot and a short paragraph. A full-width yellow manifesto block ("BUILT FOR WEB PROFESSIONALS...") in condensed caps acts as a mid-page poster, flanked by small corner labels (EST 2021 / NY – LVIV). The features area is a two-column staggered grid on sky blue: condensed uppercase heading, screenshot, short caption, repeated; one pink rounded card breaks the grid for "Custom Code" with integration logo chips. Closes with a white rounded "Get Started" card and a violet community band with phone mockup.

## Components
Buttons: two dialects — hero "TRY FREE" is a sharp-cornered black rectangle with yellow bold condensed uppercase label; closing "TRY FREE" is a bright green (#00EA40) pill with black bold uppercase; "READ OUR DEV DOCS" is a small green pill on the pink card. No shadows, no borders, flat fills only. Cards: large-radius (roughly 20-30px) flat color panels (white Get Started card, pink Custom Code card); screenshots sit in plain rectangles or minimal browser chrome. Integration logos live in small white rounded chips on the pink card. No visible inputs. Links in the footer are colored words (pink/blue) inside condensed uppercase text.

## Signature
- Pixel-glitch checkerboard logotype rendered as the hero headline
- Ultra-bold condensed uppercase manifesto blocks set solid, filling the full band width
- Stacked full-bleed pastel-plus-acid color bands (#FDFF70 yellow, #DFE4FE lavender, #93D3FD blue, #B4A1FA violet, #FEDDDA pink) with black hairline dividers
- Single electric-green pill CTA as the only saturated action color against flat fields

## Do / Don't
Do:
- Set headlines huge and near-black: condensed caps for statements, plain lowercase grotesque for one-word section titles ("Design", "Animate")
- Build the page as flat, edge-to-edge color bands and switch the band color per section
- Keep all interactive color in one bright green (#00EA40); everything else stays black-on-color
- Use thin black horizontal rules to separate repeating feature rows
- Let product screenshots be the only imagery, framed in minimal browser chrome

Don't:
- Add gradients, drop shadows, or outlined/ghost buttons — every surface here is a flat fill
- Use more than one accent hue per band; the bands themselves carry the color variety
- Set body copy large — it stays small and quiet to amplify the display-type contrast
- Round the hero CTA; the sharp black rectangle is deliberate against the pill used at page end
- Introduce decorative icons or illustration — type and color blocks do all the work
