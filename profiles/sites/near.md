## Overview
A playful, color-blocked one-pager: a clean white hero gives way to a stack of full-bleed saturated sections (forest green, teal, periwinkle purple, coral), each carrying organic blob shapes and a subtle dot-grid texture. A single white "subway line" path snakes vertically through every section, connecting rounded label tabs ("Create", "Distribute", "Discover", "Ready?") like stations. Black-on-white typography up top, white/tinted type on color below.

## Layout
Top nav is a slim white bar: search field left, center text links (Discover, Develop, Connect, Solutions), and two pill buttons right (outlined "Sign In", solid black "Create Account"). Hero is centered: large black headline with a green-highlighted word ("OS"), short paragraph, one mint pill CTA, then a monochrome logo strip ("Trusted by..."). Below, each theme section is a full-width color field with a big white tab title, a left-aligned intro block, and a 2-column grid of icon + title + paragraph + small pill-button feature items. The connecting white path enters and exits each section with rounded 90-degree bends. Footer returns to a calm cream (#f3f2e9) with a 6-column link grid and social icon row.

## Components
Buttons: fully rounded pills throughout; fills vary by context — mint green (#5beda5) for the hero CTA, black (#171813) for account CTAs, white or translucent-outline pills for secondary "Learn more →" actions on colored sections. Arrows (→) commonly suffix button labels. Section titles sit inside white rounded-rectangle tabs with the title text colored to match the section hue. Feature items are borderless, shadowless: small rounded icon tile + heading + short paragraph + pill link. Search input is a soft gray pill. No visible card borders or drop shadows anywhere; hierarchy comes from color blocks alone.

## Signature
- Vertical color-block storytelling: green → teal → purple → coral sections stacked full-bleed, each with organic blob overlays in a lighter/darker sibling hue plus a faint dot grid.
- The continuous white connector line threading the whole page, docking into white rounded label tabs that name each chapter.
- Green highlighter box (#01ec98) behind one keyword in the hero headline.
- Flat design discipline: zero shadows, zero borders — pills, blobs, and saturated fields do all the work.

## Do / Don't
Do:
- Keep the hero white and typographic, saving the loud color for the scrolled sections.
- Give each major section its own full-bleed hue and repeat the pattern: white tab title, intro block, two-column feature grid.
- Use pill buttons everywhere and vary the fill (mint, black, white/outline) by prominence, with → arrows on secondary links.
- Layer organic blob shapes and a subtle dot-grid texture in tonal variants of each section color.
- Draw the white connector path with rounded corners to physically link sections.

Don't:
- Add drop shadows, card borders, or gradients — the language is completely flat.
- Use more than one accent per section; each block stays monochromatic with tonal blobs.
- Set headlines in a serif or condensed face; everything is one geometric grotesque family.
- Put photography or screenshots in sections — imagery is limited to simple monochrome glyph icons.
- Sharpen corners: nothing on the page has a hard 90-degree corner except the color fields themselves.
