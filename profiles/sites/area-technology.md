## Overview
A near-empty white stage built around one enormous typographic wordmark and one photoreal 3D emblem. The word "AREA" is set edge-to-edge in a muted slate blue-gray, and beneath/behind it a glossy rendered object (chrome disc with green dot-matrix globe screens, or a vivid blue atom-like "A") carries all the color and depth. Everything else — tagline, two icon links — is tiny, uppercase, and the same quiet slate gray. The site reads as a gallery pedestal for experimental CGI identity work.

## Layout
No conventional nav bar. The wordmark itself is the header, spanning the full viewport width at the top. The hero is the entire page: a single centered 3D object roughly 45% of viewport width, vertically centered. Bottom-left corner holds a one-line uppercase tagline ("DESIGN STUDIO FOR ADVANCED VISUAL TECHNOLOGY"); bottom-right holds two icon-plus-label links (folder icon "PROJECTS", info icon "INFORMATION"). No sections, no scroll rhythm visible — one screen, four fixed anchor points (top, center, bottom-left, bottom-right), massive whitespace. The emblem animates through fade/opacity states (ghosted washed-out versions appear between full-color states), so motion substitutes for section rhythm.

## Components
No buttons, cards, or inputs appear on any capture. The only interactive elements are two flat icon links: simple filled glyphs (folder, circled "i") in slate gray above 10-11px uppercase labels, no borders, no background fills, no shadows, no visible hover state captured. All chrome-and-glow depth (bevels, specular highlights, emissive green LED-dot screens) lives exclusively inside the rendered emblem, never in UI chrome. Corners on everything are square.

## Signature
- Full-bleed heavy extended grotesque wordmark stretched to the exact viewport width, in desaturated slate #9BA8B7 rather than black.
- One hyperreal 3D centerpiece (chrome, glass, emissive dot-matrix green globe) as the only saturated element on an otherwise monochrome page.
- Emblem cycles through opacity/ghost states — the page breathes between washed-out and full-color renders.
- Corner-anchored micro-typography: tiny uppercase gray labels pinned to bottom-left and bottom-right, nothing else.

## Do / Don't
Do:
- Keep the canvas pure white and reserve all saturation for a single rendered object.
- Set the wordmark in a black-weight extended grotesque and scale it to the full viewport width.
- Use muted slate blue-gray (#9BA8B7) for every piece of text and iconography, including the logo type.
- Pin navigation as tiny uppercase icon-labels in the bottom corners instead of a top nav bar.
- Let the centerpiece fade between ghosted and vivid states for motion interest.

Don't:
- Add buttons, pills, borders, or drop shadows to UI — the site has zero conventional components.
- Use black or near-black text; even the display type stays mid-gray.
- Introduce more than one focal object or fill the whitespace with sections and copy.
- Round any corners; geometry is square except inside the 3D renders.
- Put color into links or labels — green and blue exist only inside the emblem artwork.
