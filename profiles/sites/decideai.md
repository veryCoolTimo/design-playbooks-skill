## Overview
Near-black technical brutalism with schematic line-art instead of photography. Every section is a hairline-bordered rectangle annotated with tiny corner labels ("FDP / VERIFICATION / 001. / AI"), like plates in an engineering document. One accent color per product page — terminal green for identity/home, signal red-orange for the Protocol page — over a strict black/off-white base. Wireframe torus and orbital SVG figures carry the "decentralized AI" theme.

## Layout
Top nav: tiny logo left, small text links right ("White Paper" underlined), on black, no background bar. Heroes are left-aligned: large grotesque headline, one-line subhead, decorative data-strip motifs (barcode/waveform graphics with monospace-style micro labels like "ST/00 01.DATA"), a geometric line diagram on the right, and an underlined "Join now ↗" CTA with "Discover ↓" opposite. Sections alternate full-black and inverted off-white card panels; a giant clipped marquee "DecideAI ✦ Decide" wordmark runs edge-to-edge as a section divider (repeated before the footer). Card grids are edge-to-edge with thin gutters; numbered lists (001. / 002. / 003.) with rule separators structure feature sections. Dense, systematic rhythm — whitespace comes from the black voids between framed plates, not from padding.

## Components
- CTAs: no filled buttons anywhere — bold underlined text links with ↗ or ↓ arrow glyphs, white on black or black on white.
- Cards: sharp rectangles, 1px light borders on black (or solid off-white fills for inverted cards), corner metadata labels in tiny caps at all four corners, zero shadow, zero radius.
- Diagrams: flowchart panel with thin-stroke boxes, green/red state labels, connector lines — an actual annotated system schematic used as content.
- Comparison table: hairline grid, checkmarks in accent green.
- Icons: thin wireframe line-art (tori, spheres, orbital rings, hooded figures), never filled illustrations.

## Signature
- Four-corner metadata annotations on every card/plate (codes like "AVT / ID / 001. / AI") — the strongest identifying tic.
- Oversized clipped marquee wordmark "DecideAI ✦ DecideAI" bleeding off both edges as a divider band.
- Barcode/spectrum data-strip graphics with monospace micro-labels replacing hero imagery.
- Per-page accent swap on an otherwise monochrome system: green (ID/home) vs red-orange (Protocol).
- Wireframe torus/loop line drawings as the sole illustration language.

## Do / Don't
**Do**
- Keep everything square: 1px hairline borders, no radius, no shadows, on near-black #0A0A0A.
- Render CTAs as bold underlined text with ↗ arrows, never filled buttons.
- Annotate cards with tiny uppercase corner labels and zero-padded numbering (001., 002.).
- Use exactly one accent color per page/section against black + off-white; reserve green for checks/states.
- Illustrate with thin-stroke wireframe geometry (tori, orbits, barcodes) and schematic diagrams.

**Don't**
- Don't add gradients, glows, glassmorphism, or rounded pill buttons — the system is flat and hard-edged.
- Don't use photography or filled/colored illustrations; all imagery is line-art.
- Don't mix the green and red accents in the same section.
- Don't loosen the density with big soft padding; sections are tightly framed plates.
- Don't set headlines in serifs or expanded tracking — display type is a tight, plain grotesque.
