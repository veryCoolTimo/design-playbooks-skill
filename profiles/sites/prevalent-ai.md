## Overview
An editorial-illustration hybrid: heavyweight serif headlines with mid-sentence violet color shifts sit over dense, hand-drawn isometric city and "floating island" scenes. The page alternates full-bleed color bands — near-black navy (#050a24), off-white, pastel mint and peach, and a saturated teal — giving a storybook rhythm unusual for enterprise cybersecurity. Illustration carries the personality; UI chrome stays minimal and flat.

## Layout
Dark navy top nav: logo left, dropdown links right, single purple pill CTA. Hero is a centered serif headline (white words alternating with violet) above a full-width detailed illustration of a navy cityscape with characters and glass domes. Sections stack as full-bleed horizontal bands that swap background color (white → pastel mint/peach feature panels → navy → white → teal → navy footer). Feature rows are two-column: text block left, floating-island spot illustration right. A three-up card row uses thin vertical hairline dividers instead of boxed cards. Generous vertical whitespace between bands; content sits in a wide centered container.

## Components
Primary buttons: solid #6360d5 pill, white text, no border or shadow. Secondary actions are underlined text links ("Learn more") with a small southeast arrow glyph. Icon tiles: ~20px-radius rounded squares filled with flat violet/peach/mint, containing outlined hand-drawn objects (glasses, keyboard, wallet). Feature panels are edge-to-edge pastel rectangles with square corners and dashed decorative path lines; a large rounded-rectangle keyline outlines the illustration zone. Resource cards are square-cornered image thumbnails with plain text below. Everything is flat — no drop shadows or gradients.

## Signature
- Floating-island illustrations: hand-drawn characters and furniture perched on levitating rock discs, one per feature section
- Serif headlines that switch to violet (#6360d5) or teal mid-sentence for the key phrase ("Security technology is broken.")
- Full-bleed band rhythm cycling near-black navy, off-white, pastel mint/peach, and teal #18c4a3
- Dashed dark-green path lines wandering through pastel panels as decorative connectors

## Do / Don't
**Do**
- Set every headline in the heavy high-contrast serif and color exactly one phrase violet or teal
- Alternate full-bleed background bands (navy #050a24 / off-white / pastel mint-peach / teal) instead of boxed sections
- Pair each feature block with one isometric hand-drawn spot illustration on a floating base
- Keep buttons as flat violet pills and secondary links as underlined text + arrow
- Separate card columns with hairline vertical rules, not borders or shadows

**Don't**
- Use drop shadows, gradients, or glassmorphism — every surface here is flat
- Set headings in a sans-serif; the serif/sans split (serif display, grotesque body) is the core contrast
- Use stock photography or 3D renders; imagery is exclusively line-and-fill illustration (photos appear only as small resource thumbnails)
- Round the corners of large section panels — big surfaces stay square; only icon tiles and buttons curve
- Introduce red/orange alert colors; even for a security product the palette stays violet-teal-pastel
