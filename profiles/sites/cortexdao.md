## Overview
Minimal-light DeFi landing built on a cool light-gray canvas with near-white cards, dark navy-charcoal panels for emphasis moments, and a single saturated green reserved for CTAs. Iridescent chrome 3D ring renders (holographic pastel gradients) supply the only decorative flourish against otherwise flat, quiet surfaces. Soft mint/green radial washes bleed into the background at section edges.

## Layout
Slim top nav: wordmark left, center text links, one pill green "Launch App" CTA right. Hero is left-aligned text (two-line headline with weight shift), small caption paragraph above the buttons' scale, dual CTAs, and a large 3D ring floating off the right edge. Sections alternate: centered oversized headline, then a staggered/offset cluster of small feature cards rather than a strict grid; mid-page a full-width dark navy feature card with embedded product chart UI; later a circular tokenomics diagram with pill nodes; page closes in a full-bleed dark navy band (token section, audit strip, footer). Generous vertical whitespace between sections; content column is narrow relative to the 1920px canvas.

## Components
Buttons: fully pill-shaped, solid #17A672 with white label (primary) or near-white #FAFAFA fill with green label (secondary); no visible borders or heavy shadows. Cards: near-white #FAFAFA on the gray canvas, 12-16px radius, extremely subtle shadow, small line icon top-left with short copy — sized to content, not equalized. Dark variant cards (#191E2C) carry white/mint type and embedded mini-UI (price chart, stat rows). Fake product widgets (Buy/Sell panel, balance card) reuse the same card language with tiny pill buttons. Pill chips with token avatars used as diagram nodes.

## Signature
- One green (#17A672) doing all interactive work — every CTA is the same pill, nothing else competes.
- Iridescent holographic 3D rings as the recurring brand object, from hero to token section.
- Staggered, asymmetric card clusters instead of aligned grids for feature/benefit content.
- Dark navy (#191E2C, not black) blocks with mint-cyan (#45E0BE) token names for contrast moments.

## Do / Don't
Do:
- Reserve #17A672 strictly for buttons and small status/positive numbers; keep everything else neutral.
- Use pill radius on every button and chip; keep card corners noticeably softer (12-16px) but not pill.
- Offset small feature cards in a loose, overlapping-column arrangement with uneven vertical starts.
- Punctuate the light page with full-width #191E2C panels for product and token moments.
- Let 3D iridescent renders bleed off the canvas edge rather than sit boxed inside sections.

Don't:
- Don't use pure white page backgrounds — the canvas is a gray #EFEFEF that makes #FAFAFA cards read as elevated.
- Don't add borders, outlines, or strong drop shadows to cards; separation comes from fill contrast alone.
- Don't introduce a second saturated hue for UI; mint-cyan appears only as text highlight on dark panels.
- Don't set headlines in a single weight — the hero mixes regular and bold within one phrase.
- Don't use pure black for dark sections or text; everything dark is the navy-tinted #191E2C.
