## Overview
A warm, quiet minimal-light page built on a single greenish off-white canvas (#FAFBF8) that never switches to hard section bands. One saturated sky blue (#3AA4EE) does all the CTA and data-visualization work, while near-black headings, soft gray body copy, and hand-drawn character illustrations keep the tone friendly rather than corporate. Density is low; the product itself (a live dashboard screenshot) is the hero artwork.

## Layout
Single-row top nav: tiny logo left, sparse text links plus avatars right, ending in one small blue pill CTA. Hero is fully centered: two-line heading, two-line subcopy, a blue primary + gray secondary button pair, then a pill tab switcher (Dashboard / Visitors / Journeys / Funnels) above a large flat dashboard screenshot. Sections alternate between centered headings with logo strips, a letter-style founder note card, illustration-led feature intros, and a 3x3 bordered feature grid. Generous vertical whitespace between sections (roughly 150-250px); content column stays narrow (~800-1000px) inside the 1920px canvas. FAQ is a plain accordion list; final CTA is a bordered card with a two-tone heading (black + gray) and an illustration.

## Components
Primary buttons: solid #3AA4EE fill, white label, small radius (~4-6px), no shadow, compact padding. Secondary buttons: pale gray fill with subtle border and a small "x"/icon suffix, same small radius. Tab switcher: pill-shaped, active tab inverted to near-black (#23241F) with white text. Cards: white or off-white fills with 1px light borders, medium radius, essentially shadowless (the founder letter card gets a faint paper shadow). Feature grid: one bordered container subdivided by hairlines into 9 cells, each with a tiny outline icon, bold mini-heading, and gray body text with inline blue links. Dashboard screenshot uses the same blue for all bar charts, reinforcing the single-accent system.

## Signature
- One blue does everything: hero CTA, nav CTA, chart bars, and inline links are all the same #3AA4EE — no gradient, no second brand color.
- Hand-drawn ink-and-wash character illustrations (woman with boxes, peeking figure) dropped into an otherwise flat UI page, with faint teal/gray sketch lines radiating around them.
- A typed founder letter styled as a physical note card with a handwritten signature — editorial trust device instead of testimonials.
- Warm near-white canvas that never changes color; sections are separated only by whitespace and hairline-bordered cards.

## Do / Don't
Do:
- Keep the entire page on one warm off-white (#FAFBF8); create structure with hairline borders and whitespace, not background color changes.
- Reserve #3AA4EE for actions and data marks; render charts in that same blue so product shots feel on-brand.
- Use the inverted near-black pill only for the active state of tab/segment controls.
- Pair every primary blue button with a quiet gray secondary ("View live demo") at the same small radius.
- Let illustrations carry personality: loose ink characters with a few scattered accent strokes, not polished 3D or stock art.

Don't:
- Don't add drop shadows to buttons or cards; the only elevation seen is a faint paper shadow on the letter card.
- Don't introduce gradients, dark section bands, or a second accent hue — the palette is blue + charcoal + paper.
- Don't use large or pill radii on buttons; pills belong to the tab switcher only.
- Don't widen body copy; text blocks stay narrow, centered, and set in muted gray rather than pure black.
- Don't crowd sections — vertical breathing room between blocks is a core part of the rhythm.
