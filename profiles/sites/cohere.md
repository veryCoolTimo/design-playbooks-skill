## Overview
Warm ecru canvas alternating with near-black tech sections, punctuated by iridescent 3D gradient blobs (coral, orchid, sage, periwinkle) that bleed off page edges. The system pairs an editorial grotesque with monospace utility labels and embedded code/terminal UI, landing between technical documentation and art-directed print. Buttons are skewed parallelograms with arrow glyphs, giving a deliberate constructed, techno-brutalist edge.

## Layout
Sticky pill-shaped nav bar (white capsule floating on the canvas) with mono uppercase links and a dark "TRY NOW" pill at right. Heroes are asymmetric: left-aligned large heading + short paragraph + stacked CTA pair, with a gradient blob or product-UI mock filling the right half. Sections alternate backgrounds in bands — cream, sage green, near-black (#0E0E0E), lavender-gradient — creating strong vertical rhythm. A dark customer-logo strip sits directly under most heroes. Content sits in a generous centered column; feature grids use 2-3 columns with big numbered items (1-4) on notched cards. Footer is a tall dark block ending in an oversized rounded-slab "cohere" wordmark spanning the full width.

## Components
Buttons: primary is a black parallelogram (skewed left edge, clipped corner) with mono uppercase label and a small arrow tab attached at the right; secondary is the same shape outlined on white. Cards: off-white panels with thin borders, small radius, and occasional diagonal-cut ("notched") corners; numbered belief cards use large serif-scale numerals. Pricing uses a saturated periwinkle-blue panel with white mono figures and stacked tab buttons. Code blocks are dark green-black editors with orange/green syntax highlighting. FAQ rows are thin-ruled accordions with mono uppercase questions and arrow icons. Inputs: dark email field with square arrow-button suffix in the footer.

## Signature
- Iridescent 3D gradient blobs (coral/orchid/sage/pearl) cropped at page edges against warm ecru
- Skewed parallelogram CTA buttons with detached arrow tab and mono uppercase labels
- Alternating cream / sage / near-black full-width section bands with embedded terminal-style code UI
- Giant rounded-slab lowercase "cohere" wordmark filling the footer

## Do / Don't
Do:
- Alternate warm cream, sage, and near-black full-bleed section bands to pace the page
- Use monospace uppercase for eyebrows, nav links, button labels, and FAQ headers
- Cut buttons as skewed parallelograms with a clipped corner and trailing arrow glyph
- Let gradient blob renders crop off the viewport edge instead of sitting fully inside sections
- Show real product UI and syntax-highlighted code blocks inside dark panels

Don't:
- Use colored CTA fills — CTAs stay black/outlined; color lives in blobs and section backgrounds only
- Use pure white or cool gray as the page background; the canvas is warm ecru
- Round buttons into pills (pill shape is reserved for the nav capsule)
- Add drop shadows or glassmorphism — depth comes from flat banding and thin borders
- Center hero copy; headings and CTAs are left-aligned against right-side visuals
